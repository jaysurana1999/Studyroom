# About The Project
Used The Rick and Morty API called just after mounting and created frontend with a List of each character having Image and other information. Also used Redux store to store the clicked data which is shown on History Page.
A swipe to move between difffrent views, the views are current,, history and form, form consist of asking the user their favourite character.

# Sample
Please refer to the sample Screen recording of both android and ios in the repository.

# Instructions to run the project

* Clone the repo
```
git clone https://github.com/jaysurana1999/Studyroom.git
```
* go to the cloned project folder
* for running on ios: 
```
npx react-native run-ios
```
* for running on android:
```
yarn android
```
